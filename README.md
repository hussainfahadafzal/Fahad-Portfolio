# Fahad Afzal Hussain — Portfolio

A modern, responsive personal portfolio website showcasing my projects, journey, and technical interests as a full-stack developer.

🌐 Live Demo: https://fahadafzalhussain.vercel.app/

## Features
- Clean and modern UI
- Responsive layout for desktop and mobile
- Sections for About, Projects, Timeline, Skills, and Contact
- Smooth animations and polished visual design
- Easy to customize and deploy

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Structure
- index.html — main page structure
- style.css — styling and animations
- script.js — interactions and UI behavior
- assets/ — images and other static assets

## Run Locally
Open index.html in your browser, or use a simple local server such as:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
